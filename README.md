### Launch guide
To launch the code install from https://www.python.org/downloads/ the latest version of python

with this you can simply double click on the main.py file and it will be executed

alternatively install to visual studio code "python debugger" and "python" extensions so you can debug from visual studio code the various lines of code

remember to leave in the same folder dati.json and main.py otherwise the code wont work

### Requirements
DLL: pathlib and json

Used python version: Python 3.14.3

### How to modify
To modify the input parameters just modify the "dati.json" file with the parameters you want: be aware to not modify the names in the json but just the values in the ListaFinale.txt the program will write down the result
